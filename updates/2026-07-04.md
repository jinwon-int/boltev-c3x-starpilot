## 🔄 변경 방법

C3X 인터넷 직결로 `git fetch` 사용 (tarball 우회 불필요). untracked 스캐폴딩 파일은 백업 후 stash 처리.

```bash
# C3X에서 (SSH 릴레이 경유)
cd /data/openpilot
# 롤백지점/백업
git rev-parse HEAD > /data/backups/preupdate-HEAD-<TS>.txt        # 6e56fff
tar -czf /data/backups/preupdate-untracked-<TS>.tar.gz -T <untracked.list>
# fetch (prebuilt 바이너리 커밋들이라 대용량 → 백그라운드 nohup 권장)
git fetch origin StarPilot
git stash push -u -m "preupdate-<TS>"
git checkout d931d3007
# prebuilt 마커 확인 후
sudo reboot
```
